# Epic User Story – CLI-Agenda
## Gestión integral de notas, tareas y eventos desde la terminal

Como usuario de CLI-Agenda, quiero poder gestionar todas mis tareas, notas y eventos importantes desde la terminal, para que pueda organizar mi vida personal y profesional de manera eficiente sin depender de aplicaciones gráficas.

### Descripción:

CLI-Agenda es una aplicación de consola escrita en Java que centraliza la gestión de información relevante para el usuario. Permite crear, editar, listar y eliminar tareas y notas, así como registrar y recibir notificaciones de eventos importantes. La herramienta ofrece filtrado y ordenamiento por prioridad, estado y fecha, y soporta repetición de eventos de manera anual o personalizada.

### Criterios de aceptación (Acceptance Criteria):

- El usuario puede crear, actualizar, listar y eliminar tareas.
- Las tareas se pueden marcar como completadas y filtrar por prioridad, estado o proximidad de fecha.
- El usuario puede crear, editar, listar y eliminar notas.
- Cada nota debe estar relacionada a una tarea y las tareas pueden tener varias notas.
- El usuario puede registrar eventos importantes, como cumpleaños, recordatorios o citas.
-  Los eventos pueden repetirse anualmente o de forma personalizada.
-  La aplicación envía notificaciones para los eventos próximos.
-  Se puede añadir tareas en los eventos.
-  Todas las funcionalidades se pueden realizar completamente desde la terminal con comandos claros.### Notas adicionales: CLI-Agenda debe ser intuitiva y rápida, priorizando la eficiencia en la terminal.
